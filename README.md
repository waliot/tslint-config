# Waliot tsling config

## Install

```bash
npm install --save-dev @waliot-oss/tslint-config
```

## Usage

`tsling.json` file in your project:

```json
{
  "extends": "@waliot-oss/tslint-config",
  "rules": {
    "no-addition": true
  }
}
```
